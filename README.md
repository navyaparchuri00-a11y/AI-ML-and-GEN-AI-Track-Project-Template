📊 A Comprehensive Measure of Well-Being
Human Development Index (HDI) Prediction System
📌 1. Project Overview

The Human Development Index (HDI) Prediction System is a machine learning-based web application designed to predict the development level of a country using socio-economic indicators such as life expectancy, education, and income.

The system integrates a trained ML model with a Flask web application to provide real-time, user-friendly predictions.

🎯 2. Objectives
To analyze key human development indicators
To build a machine learning model for HDI prediction
To develop a web-based user interface using Flask
To understand the impact of socio-economic factors on development

🧠 3. Features
Secure user login system
HDI prediction using ML model
Real-time result generation
Prediction history storage
Simple and responsive web interface
Cloud deployment support (Render)

🔄 6. System Workflow
User enters input values in the web interface
Flask backend receives and processes input
Trained ML model performs prediction
HDI category is generated
Result is displayed on the web page
Prediction is stored for history tracking
